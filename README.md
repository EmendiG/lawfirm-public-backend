# Backend for lawfirm webpage
Secured web application backend created with an usage of Java Spring Boot Framework and: <br/>
* Spring Boot Web <br/>
* Spring Security <br/>
* Spring Mail <br/>
* Spring Data JPA <br/>
* Hibertnate <br/>
* Postgresql <br/>
* Aspectj AOP <br/>
* Lombok <br/>
* Maven <br/>
